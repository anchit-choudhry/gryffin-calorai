# Release Notes - v0.0.1 (Initial Release)

Welcome to the initial release of **Gryffin Calorai**, a modern, client-side web application for managing your daily nutritional goals. This version introduces core tracking features with a focus on local privacy and high performance.

## Key Features

### 📊 Comprehensive Dashboard

- **Daily Intake Summary**: Get an immediate overview of your total calorie consumption for the current day.
- **Log History**: Review a list of items logged today, including calorie counts and serving sizes.
- **Visual Progress**: Real-time updates to your daily total as you log food.

### 🥗 Food Logging & Management

- **Manual Logging**: Quickly add food items by name, calories, and serving size.
- **Barcode Scanner (Beta)**: Integrated placeholder for barcode scanning functionality to streamline data entry in future updates.
- **Recipe Collection**: A dedicated space to browse and manage your custom food recipes.

### 📈 Progress Tracking

- **Weekly Trends**: Visualize your caloric intake patterns over time with interactive line charts powered by Chart.js.
- **Data Insights**: Easily identify days where you met or exceeded your nutritional goals.

### 🛡️ Privacy & Performance

- **Local-First Data**: All food logs, recipes, and user profiles are stored locally on your device using IndexedDB (via Dexie.js), ensuring your data remains private and accessible offline.
- **Global State Management**: Fast and predictable application state transitions using Zustand.

### 🎨 Modern User Experience

- **Responsive Design**: Built with React and Tailwind CSS for a seamless experience across desktop and mobile devices.
- **Dark Mode Support**: Automatically respects system theme preferences with manual toggle support.
- **Fast Navigation**: Hash-based routing for instantaneous page transitions without full reloads.

## Technical Details

- **Framework**: React 18+ with TypeScript
- **Styling**: Tailwind CSS v4
- **Database**: Dexie.js (IndexedDB wrapper)
- **State**: Zustand
- **Visualizations**: Chart.js with react-chartjs-2

## Getting Started

To get started, simply visit the dashboard and begin logging your first meal. No account creation is required, as all data is stored locally in your browser.
