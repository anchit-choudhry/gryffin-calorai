## Major Features & Improvements

### 🆕 New Components & Utilities

#### **Error Boundary Component** (`src/components/ErrorBoundary.tsx`)

- Graceful error handling for React runtime errors
- User-friendly error fallback UI
- Error logging with context
- Reload recovery mechanism

#### **Custom Form Hooks**

- **`src/hooks/useFoodForm.ts`** (73 lines)
  - Encapsulates food logging form state management
  - Input validation with min/max constraints
  - Form reset and error messaging
  - Async submission handling

- **`src/hooks/useRecipeForm.ts`** (98 lines)
  - Recipe creation form logic
  - Dynamic ingredient management
  - Validation for recipe metadata
  - Calorie calculation aggregation

#### **Shared Types System** (`src/types/index.ts`)

- Branded types for type-safe ID handling (UserId, FoodItemId, RecipeId, ISODate)
- Type guards for runtime validation
- Helper functions for date formatting
- Compile-time ID mix-up prevention

#### **Vitest Configuration**

- **`vitest.config.ts`** — Test runner configuration with jsdom environment
