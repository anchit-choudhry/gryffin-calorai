# v0.15.0 Release Notes

**Release Date:** June 19, 2026
**Database Schema:** v20 (frontend IndexedDB) + v4 (backend PostgreSQL via Flyway)
**Status:** Minor Release

---

## Overview

v0.15.0 closes two visual inconsistencies introduced in v0.14.0: the seasonal masthead
ornament was dashboard-only, and the achievement showcase used plain emoji instead of the
HarvestStamp seal language established in v0.9.0. The seasonal ornament is extracted into a
shared component and added to the navigation running head. Achievements are replaced with a
"Specimen Plates" gallery using the same HarvestStamp SVG seal motif from the weekly harvest
digest. No state changes, no DB schema changes, no new dependencies. Two new test files
deliver 21 additional tests with 100% branch coverage on the seasonal ornament.

---

## Seasonal Masthead

### Frontend - Complete (June 19, 2026)

**`apps/web/src/components/icons/almanac/SpringBlossom.tsx`**

- New icon component; `viewBox="0 0 48 48"`; a curved branch with three open-circle blossoms
  rendered as `<circle>` elements with stamen dots; used for the spring and early-summer date
  range in `SeasonalOrnament`

**`apps/web/src/components/icons/almanac/WinterBranch.tsx`**

- New icon component; `viewBox="0 0 48 48"`; a diagonal branch with two offshoots and four
  six-point snowflake marks drawn with short `<line>` elements; used for the winter date range

**`apps/web/src/components/icons/almanac/SeasonalOrnament.tsx`**

- New component consolidating all seasonal display logic previously duplicated between
  `DashboardHero.tsx` and other locations
- `isNearTransition` check fires first: if `date` is within 4 days of any solstice or
  equinox, renders `SeasonalFlourish` (the existing transitional ornament)
- Otherwise: Spring (Mar 25-Jun 16), Summer (Jun 26-Sep 17), Autumn (Sep 27-Dec 16), Winter
  (Dec 26-Mar 15) mapped to `SpringBlossom`, `SummerWheat`, `AutumnLeaf`, `WinterBranch`
- Exported from `apps/web/src/components/icons/almanac/index.ts`

**`apps/web/src/components/dashboard/DashboardHero.tsx`**

- `computeLiveMoonPhase`, `NEW_MOON_REF_MS`, and `LUNAR_CYCLE_MS` removed; moon phase now
  sourced from `getMoonPhase(today).phase` in `solar.ts` (astronomically correct JDN formula
  with 15 existing tests)
- `SeasonalOrnament` replaces the inline seasonal icon switch; no behavioral change for the
  user

**`apps/web/src/App.tsx`**

- `<SeasonalOrnament date={new Date()} />` placed left of the "Field Journal" text in the
  `hidden md:flex` navigation running-head strip; sized `h-3 w-auto opacity-50` so it reads
  as a secondary mark without competing with the title

## Specimen Plates

### Frontend - Complete (June 19, 2026)

**`apps/web/src/components/progress/SpecimenPlate.tsx`**

- New component wrapping the HarvestStamp SVG seal pattern: outer dashed ring (`r=42`,
  `strokeDasharray="3 4"`), solid ring (`r=36`), hairline inner ring (`r=30`,
  `opacity=0.5`); designed to match the `HarvestStamp` used in the weekly harvest digest
- Earned plates show the achievement emoji, a mono plate number (`Plate I`, `Plate II`...
  via `toRoman()` from `utils.ts`), and the unlock date in persimmon below the seal
- Unearned plates are wrapped in `opacity-[0.18]` with `filter: grayscale(1)` applied to the
  emoji so they read as faint blind-embossed outlines rather than locked padlock icons

**`apps/web/src/pages/Progress.tsx`**

- The two-branch `ACHIEVEMENTS.map()` block (~40 lines with earned/unearned inline JSX)
  collapsed to a single `<SpecimenPlate achievement={a} unlocked={unlockedMap.get(a.id)} />`
  call per item
- Section title updated from "Achievements" to "Specimen Plates" to match the design idiom
- Achievement grid updated to `grid-cols-2 sm:grid-cols-3 md:grid-cols-4 lg:grid-cols-6`
  for a periodical-index layout on large screens
- `toRoman()` and `unlockedMap` lookups retained in the parent component; `SpecimenPlate`
  receives a pre-resolved `unlocked?: UserAchievement` prop

## Upgrade Notes

No schema migrations required. All changes are additive:

- Seasonal masthead ornament now appears in the navigation running head on medium-and-above
  screens; it was previously only visible in the Dashboard Hero band
- Achievement showcase is now labeled "Specimen Plates" and renders the HarvestStamp seal;
  achievement state and unlock logic are unchanged
