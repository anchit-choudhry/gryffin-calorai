# v0.6.0 Release Notes

**Release Date:** May 25, 2026
**Database Schema:** v17
**Status:** Major Feature Release

---

## Overview

v0.6.0 delivers meal planning and templates (Feature 16), per-recipe macro breakdowns, advanced
recipe search, meal-type filtering on the Dashboard log, richer achievement toast feedback, and
recipe description sanitization. This is the current stable release.

---

## Major Features

### Feature 16: Meal Planning & Templates

Save named templates from any day's log and schedule a 7-day meal plan.

**What's New:**

- `MealTemplate` entity: snapshot of food items saved under a name; `MealTemplateId` branded type
- `MealPlan` entity: 7-day array of `{ dayIndex, templateId | null }` assignments; `MealPlanId` type
- DB schema v17: `mealTemplates` and `mealPlans` tables with compound indices on `[userId]`
- `src/state/AppState.ts`: `mealTemplates`, `mealPlans` state; `addMealTemplate`,
  `deleteMealTemplate`, `saveTemplateFromTodayLogs`, `logAllFoodsFromTemplate`,
  `copyFoodsFromDate`, `saveMealPlan`, `deleteMealPlan`, `applyWeekPlan` actions
- `src/hooks/useMealTemplates.ts`: react-hook-form + zod, exposes all template and plan actions
- `src/components/MealTemplates.tsx`:
  - "Save Today" path: save today's food log as a named template
  - "Copy Yesterday" button: copies yesterday's food log to today
  - Create blank template via name form with FormField validation
  - 7-day plan creation grid: per-day template selector + plan name input + Save Plan button
  - Template list with "Log All" and "Delete" per entry
  - Plan list with "Apply Plan" and "Delete" per entry
- `applyWeekPlan` logs all template foods for each day of the current week
- `MealTemplateSchema` and `MealPlanSchema` added to `src/forms/schemas.ts`
- Integrated into `src/pages/Dashboard.tsx`

---

### Recipe Macro Breakdown

Optional per-recipe protein, carbs, and fat totals displayed on RecipeRow.

**What's New:**

- `totalProtein`, `totalCarbs`, `totalFat` optional fields added to `Recipe` type
- `IngredientSchema` extended with per-ingredient `protein`, `carbs`, `fat` fields
- `RecipeRow` displays a P/C/F macro summary line when totals are present
- `IngredientRow` auto-fills macros from the matched food item on selection

---

### Advanced Search on Recipes Page

Live name-filter input on the Recipes page.

**What's New:**

- `searchQuery` state + `useMemo` filter in `src/pages/Recipes.tsx`
- Case-insensitive name matching; instant results on keystroke
- `RecipesHero` search input using `EDITORIAL_INPUT_CLS`

---

### Meal-Type Filter on Dashboard Today's Log

Filter today's log to a single meal type via chip row.

**What's New:**

- Chip row (All / Breakfast / Lunch / Snacks / Dinner) below the Today's Log section header
- `logFilter` state in `src/pages/Dashboard.tsx`; filtered via `groupLogsByMeal()` with `useMemo`
- Selected chip highlighted; resets to "All" on date change

---

### Achievement Toast Polish

Richer achievement unlock notifications.

**What's New:**

- Sonner `description` field used alongside the title for achievement toasts
- Duration increased to 5 seconds (from 3 s) for achievement unlocks
- Applied in `evaluateAchievements` toast calls in `src/state/AppState.ts`

---

### Recipe Description Sanitization

Strip HTML tags from imported recipe descriptions.

**What's New:**

- `stripHtml()` utility in `src/lib/utils.ts`: removes all tags and script/style content
- Applied in `parseRecipeFromHtml()` in `src/hooks/useRecipeImport.ts` to clean imported
  descriptions before populating the recipe form
- Tests added in `utils.test.ts` for `stripHtml` edge cases
